Welcome to Auto Claude

Auto Claude is a multi-session Claude Code dashboard that lets you run and monitor multiple AI coding sessions simultaneously.

Auto Claude wraps the Claude Code CLI and gives you a visual interface to manage sessions, monitor token usage, track costs, and more.

Step 1: Select a Project

Click Select Project in the toolbar to choose a project folder. This is the working directory Claude Code will operate in.

Tip: You can have different projects open in different tabs. Click the + button next to the tabs to add a new session tab.

Step 2: Start a Session

Type your prompt in the input box and click Start Session (or press Ctrl+Enter). Claude Code will begin working on your request.

You can also choose between New session, Continue (resume last session), or pick a previous session to resume.

The session dropdown lets you choose session mode. "New session" starts fresh, "Continue" picks up where you left off.

Step 3: Monitor Progress

While a session runs, you'll see real-time stats:

State - Current activity (thinking, writing, tool use)
Tokens - Input/output token counts
Cost - Estimated USD cost
Tools - Number of tool calls
Model - Which Claude model is active

Click Stop to end a session early. The output panel shows Claude's full response in real-time.

Step 4: Multi-Tab Workflow

Auto Claude supports multiple concurrent sessions. Each tab runs an independent Claude Code instance.

Use the layout buttons in the top-right to view tabs as:
Single - One tab at a time
Side by side - Two tabs horizontally
Stacked - Two tabs vertically
Grid - All tabs in a grid

Settings & Configuration

Click Settings to configure:

Model - Choose which Claude model to use
Effort - Set reasoning effort level
Full Auto - Toggle auto-permission mode
Telegram - Set up notification bots
Plugins - Manage Claude Code plugins

Settings are saved automatically and persist across restarts.
